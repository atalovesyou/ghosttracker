# GhostTracker — Proposal Analytics for Freelancers

**Know when clients open your proposals, how long they read, and if they forward them.** Proposal tracking + AI follow-up emails.

🔗 **Live:** [https://atalovesyou.github.io/ghosttracker/](https://atalovesyou.github.io/ghosttracker/)

## The Problem

You send a proposal and hear nothing. Did they open it? Did they forward it to a competitor? Are they ghosting you? Freelancers lose $10K+/year to proposal silence.

## Features

- **Proposal Tracking** — Create tracked proposals with simulated real-time activity (opens, page views, forwards)
- **Engagement Scoring** — 0-100 score: Hot 🔥 / Warm / Cool / Cold
- **Page Heatmap** — Visual engagement breakdown by page
- **Activity Feed** — Chronological event log with timestamps
- **AI Follow-Up Emails** — 3 tones: gentle, value-add, urgency
- **Ghost Detection** — Auto-flag proposals going cold
- **Dashboard** — Summary stats, engagement trends, client leaderboard
- **100% Client-Side** — localStorage persistence, no data leaves browser
